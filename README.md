Digit Recognition

This project is a simple handwritten digit recognition built using Neural Network.
The model is trained on the MNIST dataset to recognize digits from 0 to 9.
It uses Python along with libraries like NumPy,Tensorflow,Keras and Matplotlib.
It also demonstrates how predictions are made and compared with actual labels.
This helps in understanding the basic concepts of machine learning and deep learning, including neural networks and classification problems.
This project was created as part of learning and practicing machine learning concepts.

Each image is 28x28 grayscale pixel.
